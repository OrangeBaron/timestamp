# TimeStamp.sh

**TimeStamp.sh** è uno script Bash progettato per aggiornare in modo automatico e ricorsivo la data di modifica delle cartelle, allineandola a quella del file o della sottocartella più recente presente al loro interno.

---

## ✨ Funzionalità Principali

* **Doppia Modalità d'Uso:**
  * *Interattiva*: Avvia lo script e trascina la cartella direttamente nel terminale.
  * *CLI (Command Line Interface)*: Passa il percorso della cartella direttamente come argomento per l'automazione.
* **Ignora File di Sistema:** Esclude automaticamente i `.DS_Store` di macOS.

---

## 🚀 Installazione e Configurazione

1. [Scarica](https://raw.githubusercontent.com/OrangeBaron/timestamp/refs/heads/main/TimeStamp.sh) o clona lo script sul tuo computer.
2. Apri il terminale e naviga fino alla cartella in cui hai salvato il file.
3. Rendi lo script eseguibile con il seguente comando:

   ```bash
   chmod +x TimeStamp.sh
